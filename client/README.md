# Frontend built with React, Vite, and Redux Toolkit.

## Setup
1. Create a `.env` file referencing `.env.example` (or use the one below for local):
    ```env
    VITE_API_URL=http://localhost:5000/api
    VITE_SOCKET_URL=http://localhost:5000
    ```
2. Run `npm install`
3. Run `npm run dev`

## Build
```bash
npm run build
```
